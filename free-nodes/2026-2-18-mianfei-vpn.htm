为什么我们需要Shadowrocket？

互联网审查制度的存在，使得许多国际知名的网站和服务（如Google、Facebook、YouTube等）在某些地区无法直接访问。这不仅限制了信息的自由流通，也影响了用户的日常工作和学习。Shadowrocket的出现，为用户提供了一种高效、稳定的翻墙解决方案。

1. 隐藏真实IP，保护隐私

通过Shadowrocket连接代理服务器，用户的真实IP地址将被隐藏，取而代之的是代理服务器的IP。这一功能不仅能够绕过地域限制，还能有效防止网络追踪，保护个人隐私。

2. 加密网络流量，提升安全性

Shadowrocket支持多种加密协议（如Shadowsocks、Vmess等），能够对用户的网络流量进行加密传输。这意味着即使在公共Wi-Fi环境下，用户的数据也不会轻易被黑客窃取或监听。

3. 突破内容封锁，畅游全球互联网

无论是学术研究、商业资讯，还是娱乐内容，Shadowrocket都能帮助用户轻松访问被封锁的网站和服务，真正实现“网络无国界”。

Shadowrocket的核心功能

Shadowrocket兼容多种代理协议，包括但不限于：

用户可以根据自己的网络环境和服务提供商选择合适的协议，以获得最佳的上网体验。

2. 自定义规则，灵活配置

Shadowrocket允许用户自定义代理规则，例如：

- 通过脚本实现高级功能（如广告过滤）。

这种灵活性使得Shadowrocket不仅适用于普通用户，也能满足技术爱好者的高阶需求。

FAQ

Netch 全局代理与规则代理的区别是什么？
全局代理会将所有流量通过节点转发，适合游戏或全局科学上网；规则代理只针对指定应用或 IP 走节点，优化带宽利用和网络效率，根据不同需求选择使用模式。
VMess 的加密方式如何选择？
VMess 支持自动加密和 AEAD 加密等多种方式。AEAD 加密更安全且抗重放攻击，适合高安全性需求的网络环境，而普通加密在性能上稍快，但安全性略低。
Quantumult X 的“分流规则”如何管理？
Quantumult X 使用策略组管理分流规则。用户可将规则文件导入应用，通过“policy group”定义不同流量走向，实现国内直连、国外代理、广告拦截等功能的自动分配。
Mellow TUN 模式适合哪些应用？
TUN 模式拦截系统 IP 层流量，实现全局代理。适合游戏、视频、浏览器等全局应用，保证所有流量走节点，提高网络稳定性和速度，比普通 SOCKS5 代理覆盖范围更广。
WinXray 如何导入订阅？
在 WinXray 界面中点击“导入订阅”选项，将机场提供的订阅链接粘贴进去。点击更新后，WinXray 会自动下载节点配置并生成列表，用户可手动选择或自动切换延迟最低的节点使用。
QuantumultX 如何通过策略组优化多应用分流？
QuantumultX 可以根据应用类型、端口和协议设置策略组，自动选择最优节点处理不同流量。这样可保证网页浏览、视频播放和游戏访问顺畅，提高整体网络效率。
VMess AEAD 加密如何保证数据安全？
AEAD 加密防止数据篡改、重放攻击和流量特征泄露。在高封锁环境下使用，可保证节点长期可用，提升数据安全和连接稳定性，同时降低被检测风险。
Surfboard 与 Clash 有什么关系？
Surfboard 是一款基于 Clash 核心的 iOS 应用，专为苹果用户设计。它通过友好的界面实现分流管理、节点切换和延迟测试等功能。与 Clash 类似，Surfboard 也支持订阅更新和多种代理协议，兼容性非常好。
VMess 节点的 alterId 参数还需要设置吗？
在新版 V2Ray/Xray 内核中，alterId 已被废弃，不再影响认证逻辑。建议用户设置为 0 或删除该字段，以避免连接异常或不兼容情况。
Hysteria2 多路复用对即时通讯的优势是什么？
多路复用可让多个短连接共享同一通道，减少握手次数和加密开销，提高吞吐量和连接稳定性。适合即时通讯应用，保证消息快速传输和稳定接收。
